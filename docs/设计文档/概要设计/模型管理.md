## ✅ 模型模块核心用例列表（Model Domain Use Cases）

| 用例编号    | 用例名称      | 描述                                       |
|---------|-----------|------------------------------------------|
| UC-M-01 | 创建模型      | 用户基于内置或自定义方式，配置并注册一个新模型。                 |
| UC-M-02 | 查看模型详情    | 展示模型的基本信息、配置参数、状态、所属租户等。                 |
| UC-M-03 | 更新模型配置    | 修改模型名称、温度、API Key、endpoint 等。            |
| UC-M-04 | 删除模型      | 删除某个不再使用的模型（通常做逻辑删除）。                    |
| UC-M-05 | 测试模型调用    | 提供测试面板，验证模型是否可用并查看返回结果。                  |
| UC-M-06 | 设置默认模型    | 在某个租户或工作区中设置默认调用的模型。                     |
| UC-M-07 | 启用 / 禁用模型 | 控制模型在平台中是否可用于调用。                         |
| UC-M-08 | 模型调用统计    | 查看模型的调用次数、平均响应时间、错误率等。                   |
| UC-M-09 | 模型绑定工作区   | 将模型关联到指定工作区或应用，控制其使用范围。                  |
| UC-M-10 | 模型能力标签配置  | 配置模型支持的功能类型，如聊天、文本生成、嵌入生成等。              |
| UC-M-11 | 管理模型供应商   | 维护支持的模型服务商（如 OpenAI、Azure、Google、阿里通义等）。 |
| UC-M-12 | 权限控制模型可见性 | 控制哪些角色/用户可以访问/配置某些模型。                    |
| UC-M-13 | 克隆模型配置    | 快速复制已有模型的配置用于创建新模型。                      |
| UC-M-14 | 模型授权共享    | 授权其他租户或用户临时使用某个模型实例。                     |
| UC-M-15 | 检测模型可用性   | 后台定期探测模型是否在线并记录状态。                       |

## 🌐 进阶用例（可选增强）

| 用例编号    | 用例名称      | 描述                              |
|---------|-----------|---------------------------------|
| UC-M-16 | 版本管理      | 支持不同版本的模型配置，如使用 OpenAI 的不同模型版本。 |
| UC-M-17 | 自动化模型接入流程 | 接入新的模型服务商后，系统自动拉取模型列表进行注册。      |
| UC-M-18 | 多模型集群路由   | 针对同一个模型类型，进行 AB Test 或流量分发策略配置。 |
| UC-M-19 | 审计日志      | 所有模型变更操作记录操作人、时间、操作内容。          |
| UC-M-20 | 模型调用限流    | 针对不同模型设置 QPS / Token 数限制。       |

---

## 🧩 用例分组建议（适配 DDD 聚合边界）

* **模型聚合（Model Aggregate）**

    * 模型本体（Model）
    * 供应商信息（Provider）
    * 模型配置参数（ModelConfig）
    * 绑定信息（WorkspaceModelBinding）

* **模型管理服务**

    * 模型注册服务
    * 模型调用验证服务
    * 模型状态监控服务
